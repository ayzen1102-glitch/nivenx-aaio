# AeroX Bot Collection

A collection of open-source Discord bots built by AeroX Development, covering security, moderation, community management, and utility use cases.

## Project Structure

This repository contains 15 standalone Discord bot projects:

- **AeroX-Security-Bot-main** — Antinuke/antiraid security bot (Node.js, discord.js 14)
- **Appy-Bot-main** — Application, ticket, giveaway, and poll bot (Node.js)
- **Container-Builder-main** — Interactive Components V2 builder bot (Python)
- **Falcron-main** — Invite tracking and giveaway bot (Node.js)
- **Feedback-Bot-main** — Structured feedback collection bot (Node.js)
- **Groove-Music-main** — Music bot with sharding (Node.js)
- **Guardian-Moderation-Bot-main** — Full moderation suite (Node.js)
- **GuildTag-Bot-main** — Guild tag role auto-assign (Python)
- **Join2Create-Bot-main** — Temporary voice channel bot (Python)
- **MC-AFK-BOT-main** — Minecraft AFK controller via Discord (Node.js)
- **ModMail-Bot-main** — DM-based modmail system (Node.js)
- **PFP-AutoPoster-main** — Profile picture auto-poster (Java)
- **Status-Bot-main** — Minecraft server status bot (Node.js)
- **Suggestions-Bot-main** — Suggestion system with Components V2 (Node.js)
- **TicketBot-main** — Support ticket system (Node.js)
- **VanityRoles-Bot-main** — Vanity status role auto-assign (Python)
- **Verifier-Bot-main** — YouTube subscription verifier via Gemini AI (Node.js)
- **Yuna-CV2-AIO-main** — All-in-One Python bot (Python)

## Dashboard

The `dashboard/` directory contains a simple Node.js web server that displays an overview of all bots.

## Running

The dashboard runs on port 5000 via `node dashboard/server.js`.
