Quick Start — All combined code is in the NivenX/ folder:
1. cd NivenX  →  npm install
2. Copy .env.example → .env and add your DISCORD_TOKEN & DISCORD_CLIENT_ID
3. npm start  ·  or  npm run start:cluster for sharding
See NivenX/README.md for full setup guide and command reference.
